## Problem statement
Complaints are often handled manually, which makes tracking and solving them difficult. Users may not know the current status of their complaint, and admins may find it hard to manage everything properly. Digital Complaint Management System helps users submit complaints, officers resolve them, and admins assign, monitor, and escalate complaints easily.

## How to Run
1. Open the project folder in VS Code .
2. Make sure the Java file is saved as: ComplaintManagementSystem.java
3. Open new terminal
4. Compile the program: javac ComplaintManagementSystem.java
5. Run the program: java ComplaintManagementSystem
